Telepítés:
1) Másold a mappa tartalmát XAMPP 'htdocs' könyvtárába (pl. C:\xampp\htdocs\projectmvp_clean)
2) Indítsd el Apache és MySQL szolgáltatásokat
3) Nyisd meg phpMyAdmin (http://localhost/phpmyadmin) és importáld az 'install.sql' fájlt
4) Nyisd meg az oldalt: http://localhost/projectmvp_clean/index.php
